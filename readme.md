# Аналіз методу Монте-Карло (Завдання 7)

## Опис завдання
Метою завдання було перевірити точність методу Монте-Карло. Ми симулювали кидання двох ігрових кубиків 1,000,000 разів, щоб обчислити ймовірність кожної суми (від 2 до 12) та порівняти її з математично розрахованою (аналітичною) ймовірністю.

## Висновки
Метод Монте-Карло показав дуже високу точність. При великій кількості ітерацій (1 мільйон) результати симуляції майже ідентичні аналітичним розрахункам.

1. **Правильність розрахунків:** Отримані відсоткові значення повністю відповідають теоретичним очікуванням (наприклад, сума 7 випадає найчастіше — близько 16.67%, а суми 2 та 12 — найрідше).
2. **Закон великих чисел:** Експеримент підтверджує, що при збільшенні кількості спроб емпірична ймовірність прагне до теоретичної. Різниця становить лише соті частки відсотка.

**Вердикт:** Метод Монте-Карло є надійним інструментом для обчислення ймовірностей у складних системах, де аналітичний розрахунок може бути важким або неможливим.
